# Project Report

# Title: Smart Expense Analyzer

# Introduction  
Managing expenses is difficult for students.

# Problem Statement  
Lack of awareness about spending patterns.

# Proposed Solution  
ML-based categorization + analysis tool.

# Methodology  
- Dataset: Custom  
- NLP preprocessing  
- TF-IDF vectorization  
- Naive Bayes classification  

# Implementation  
- Input → classify → analyze → visualize  

# Challenges  
- Small dataset  
- Limited categories  

# Results  
- Correct categorization for basic inputs  
- Clear spending insights  

# Conclusion  
Learned NLP, classification, and visualization.
